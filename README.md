# Dependencias instaladas / Librerías utilizadas

El proyecto fue desarrollado utilizando **React** en conjunto con **Vite**, lo que permitió un entorno de desarrollo ágil, moderno y eficiente.

Para el diseño de interfaces se empleó **Tailwind CSS**, facilitando la creación de estilos consistentes, modernos y adaptables a distintos dispositivos.

En cuanto a la implementación de íconos, se utilizaron las librerías **lucide-react** y **react-icons (fa6)**, brindando una amplia variedad de recursos visuales integrables en la interfaz.

La navegación dentro de la aplicación se gestionó mediante **react-router-dom**, permitiendo una transición fluida entre vistas sin necesidad de recargar la página.

Finalmente, para la representación de datos se utilizó **Recharts**, proporcionando herramientas para la creación de gráficos claros, interactivos y personalizables.
